##객체

  • 배열은 순서가 있는 리스트형이고, 객체는 Key, value 구조의 자료구조
  • JavaScript로 데이터를 표현하기 위해서는 Array, Object를 사용.
  • Object형태는 {}로 그자룔르 표현하며, 서버와 클라이언트 간에 데이터를 교환할때 Object포맷과 비슷한 방법으로 데이터를 보냄.
