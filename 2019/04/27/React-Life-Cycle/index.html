React Life Cycle

Mounting
React 컴포넌트의 인스턴스가 생성되고 DOM에 삽입되는 과정의 life cycle

  • constructor
  • componentWillMount
  • render
  • componenetDidMount

Updating
prop또는 state의 변경으로 해당 컴포넌트가 re-render 되는 경우의 life cycle

  • Props Change

    • componentWillReceiveProps
    • shouldComponentUpdate
    • componentWillUpdate
    • render
    • componentDidUpdate

  • State Change

    • shouldComponentUpdate
    • componentWillUpdate
    • render
    • componentDidUpdate
